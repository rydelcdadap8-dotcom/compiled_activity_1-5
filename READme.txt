Repository Description

This repository contains 5 system activities for Application Development and SIA 2. Each activity is organized in separate folders for proper structure and easy navigation. It showcases different system designs and implementations developed during the course.

Activities Included
Activity 1 – Smart Decision Assistant System

A web-based system designed to help users make better decisions by evaluating options, analyzing inputs, and suggesting possible outcomes. It improves decision-making efficiency by providing structured guidance and organized data comparison.

Activity 2 – Wishlist Tracker System

A system that allows users to create, manage, and monitor their wishlist items. It helps users track desired products, goals, or items they plan to achieve or purchase in the future.

Activity 3 – Random Challenge Generator System

A web-based system that generates random challenges for users. It is designed for fun, engagement, and activity-based tasks where users receive random prompts or challenges to complete.

Activity 4 – Custom Cake Order System

A system that allows customers to place custom cake orders by selecting designs, flavors, sizes, and special requests. It improves communication between customers and bakers and ensures organized order processing.

Activity 5 – Integration App System

The Integration App System is a web-based application designed to combine and connect multiple systems or modules into one unified platform. It allows different functions or features to work together smoothly, improving data sharing, communication, and overall system efficiency. The system helps users manage processes in one centralized interface, reducing duplication of work and making operations faster, more organized, and more reliable.
